# auto-setup

This Bash script setup software in Debian Distribution.   

``` git clone https://github.com/aymericcucherousset/auto-setup.git | sh ./auto-setup/start.sh```   

Made for Debian 11

## Softwares Supported ##

| Softwares | Debian|
| :-------: | ----- |
| Apache2   | Yes   |
| Composer  | Yes   |
| Mariadb   | Yes   |
| Php       | Yes   |
| Sqlite3   | Yes   |
| Symfony   | Yes   |
