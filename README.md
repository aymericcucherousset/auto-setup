# auto-setup

This Bash script setup software in Debian Distribution.   

``` git clone https://github.com/aymericcucherousset/auto-setup.git && cd auto-setup```   
Adding rights :   
``` chmod +x ./start.sh```  
Starting installation program :   
``` ./start.sh```  

Made for Debian 11

## Softwares Supported ##

| Softwares | Debian|
| :-------: | ----- |
| Apache2   | Yes   |
| Composer  | Yes   |
| Mariadb   | Yes   |
| Php       | Yes   |
| Sqlite3   | Yes   |
| Symfony   | Yes   |
